PROJECT DESCRIPTION:

This is an example RESTful web application developed in Python using the Flask framework.
The GithubStars app connects to the Github API to read the json data for the Python
repositories which have the most stars. This data is then stored in a MySQL database
instance and displayed as a list. Please click on the "Top Python repos from DB" button
to see this list.The Navigation allows the user to click on each item in the list to see the details.
The "Top Python repos from github API" button allows you to see the data directly from the github site

Minimum Software Requirements:
  • WINDOWS 7
  • FLASK V 1.1
  • PYTHON V 3.2.8
  • MYSQL DATABASE V 8.0
